Method in one minute

Utility(scene, profile) = Σ weight(scene, feature) × feature(profile). Features used: intelligibility, noise control, comfort, loudness stability, speech prominence.

Feedback update modifies only the selected scene's utility weights with bounded shift + regularization, then verifies non-target score drift stays below threshold.

Illustrative Sonova-facing prototype, not a production hearing algorithm.
